## Write a program to check website Healthcheck Report API

### How to build

```shell
docker build -t go-healthcheck .
```

### How to run local

```shell
go run main.go
```

### How to run with container

```shell
docker run -it --rm --name go-healthcheck go-healthcheck
```

### How to run test

```shell
go test -cover
```
